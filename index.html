How-To Guide: Steam Web API

By Josh Erickson

Welcome Background info on Steam and its API

Steam is a digital distribution platform that is developed by Valve Corporation. It offers multiplayer gaming and social networking services. Steam provides the user with easy installation and updating of games on multiple computers, along with features like friends lists and groups, cloud saving, voice and text chat functionality.

Steam Web API lets you retrieve information from Steam to put on your own site or app. The information you can retrieve is generally related to specific players or games, such as profile names, game news, and player summaries.
